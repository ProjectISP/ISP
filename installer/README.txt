# INSTALLATION FOR DEBIAN x86_64 

=> Previous requirements:

    - Conda or Miniconda must be installed on the system.
    - We must be in the < base > environment.
    
=> Steps to install ISP:
    
    1 - Open a new terminal and activate the Conda < base > environment.
    2 - Navigate to the installer folder < [..]/ISP/installer/debian >
    3 - Run withoud sudo the installer script: < bash ISP_installer-debian_x86_64.sh >
    4 - Follow the steps of the ISP installer.
