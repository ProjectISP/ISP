ARF

The Array Response Function (ARF) helps to evaluate the resolution of the array in a specific bandwidth.

First you need to create a coordinates files and then load it. To create the coordinate file go to

“Files à Stations Coordinates” and fill the table with lat (o), lon (o) and depth (km).

Then you can compute the ARF and plot the map with the array stations.
