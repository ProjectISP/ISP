Shortcuts

The shortcuts are in Table 1 and in Table 2 is shown specific shortcuts for compute  spectral analysis.

 

Table 1

Shortcut

Action

'Ctrl+L'

Open Macro

'Ctrl+O'

View Picks

'Ctrl+I'

Deselect all

'Ctrl+J'

Deselect a wave (Noise, Body, Surface)

'Ctrl+C'

Clean events detected

'Ctrl+M'

Open magnitudes calculator

'Ctrl+P'

Open phases comboBox

'Ctrl+K'

Save CFs (STA/LTA, Wavelet CF, Envelope, Entropy…)

'Ctrl+N'

plot_all_seismograms

'Ctrl+B'

stack_all_seismograms

 

Table 2

Shortcut

Action

'a'

Compute individual spectrum

's'

Compute all spectrums

'z'

Compute Spectrogram
