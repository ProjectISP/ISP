MACROS

 

With ISP you can build macros to process all your available data. A macro is a list of signal processes that will be done in a specific order. The user can find the Macro window on all of the modules.

 

Here you have the list with all possibilities:

  • Remove Mean
  • Taper (windowing)
  • Normalize
  • Filter
  • Resample
  • Fill Gaps
  • Differenciate
  • Integrate
  • Time Shift (Shift the starttime of traces)
  • Add White Noise (in accordance to the maximum power of the signal)
  • Spectral Withening
  • Time Normalization
  • Remove intrumental response
  • Discrete Wavelet Transform Denoise
  • Remove spikes
